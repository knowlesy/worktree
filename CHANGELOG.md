# Changelog

All notable changes to the "git-worktrees-native" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-17

### Added
- Initial release of Git Worktrees Native.
- Added Tree View in the Source Control (SCM) panel to list repositories and worktrees.
- Added `Git Worktrees: Add Worktree` command with safe input validation.
- Added `Git Worktrees: Remove Worktree` command.
- Added `Git Worktrees: Switch Worktree` command.
- Multi-repository workspace support.
- Fully tested core Git parsing logic.
- Integration test suite for robust end-to-end verification.
